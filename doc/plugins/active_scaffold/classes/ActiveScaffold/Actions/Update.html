do_update()

A complex method to update a record. The complexity comes from the support for subforms, and saving associated records. If you want to customize this algorithm, consider using the before_update_save callback

[Source]

    # File vendor/plugins/active_scaffold/lib/active_scaffold/actions/update.rb, line 75
75:     def do_update
76:       do_edit
77:       begin
78:         active_scaffold_config.model.transaction do
79:           @record = update_record_from_params(@record, active_scaffold_config.update.columns, params[:record])
80:           before_update_save(@record)
81:           self.successful = [@record.valid?, @record.associated_valid?].all? {|v| v == true} # this syntax avoids a short-circuit
82:           if successful?
83:             @record.save! and @record.save_associated!
84:             after_update_save(@record)
85:           end
86:         end
87:       rescue ActiveRecord::RecordInvalid
88:       rescue ActiveRecord::StaleObjectError
89:         @record.errors.add_to_base as_(:version_inconsistency)
90:         self.successful=false
91:       end
92:     end
